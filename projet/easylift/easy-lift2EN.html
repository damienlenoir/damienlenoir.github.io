Features

I aimed to develop a web platform to help people who search lifts and those who can offer lifts to meet. Hence the website needed to include the following features:


  • Registration with email check
  • Log in / Log Out
  • Member space with Avatar, personnal informations, saved locations etc
  • Mailing system

A Offer/Search system

  • Register a new offer in the data base
  • Search offer with dates and localisation
  • Reply to offer via the mailing system

Langages used

To achieve the above criteria, I choosed to use the usual web langages: HTML, CSS and Javascript. For a part of the javascript, I decided to use the Jquery Framework, speecificaly Jquery "UI" which has some great feature easy to implement

The project needed a data base management system and a web serveur. I coded this part with PHP and MySQL.
Firstly I developed the site localy using LAMP, then WAMP.
