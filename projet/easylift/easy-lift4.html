Enregistrement de locations personnalisées

Je voulais que les utilisateurs puissent enregistrées de locations personnalisées car c'est plus précis et plus pratique que des noms de villes. Pour cela j'ai utilisé 2 APIs de google Maps:

Maps Embed API

Cette API permet d'inclure des cartes, le site va s'en servire pour aider l'utilisateur à trouver sa location

https://developers.google.com/maps/documentation/embed/?hl=fr
Geocoding API

Cette API permet d'obtenir les coordonées géographique à partir d'une adresse. Le site créé une requête HTTP qui retourne une réponse XML. Avec PHP on retourne les valeurs afin de les enregistrer dans la BDD via une requête SQL.

https://developers.google.com/maps/documentation/geocoding/?hl=fr
